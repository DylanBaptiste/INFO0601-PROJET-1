# INFO0601-PROJET-1

Création de carte et simulation de neige qui tombe avec ncurses

## Utilisation

- Générer les sources: 
    ```c
    $> make
    ```
- Créé une carte: 
    ```c
    $> ./neige -N decor.bin
    ```
- Lancer/Reprendre une simulation: 
    ```c
    $> ./neige -S decor.sim
    ```
- Netoyer le dossier: 
    ```c
    $> make clean
    ```
